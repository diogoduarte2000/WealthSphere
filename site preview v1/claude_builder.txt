📋 Prompt Completo para Desenvolvimento
Nome sugerido: WealthSphere ou Finvest ou Kairos

VISÃO GERAL
Plataforma web full-stack de gestão financeira pessoal, mercados e comunidade. Combina ferramentas privadas de finanças pessoais com um fórum público de discussão e acesso a mercados tradicionais e alternativos. Inspiração visual: Anthropic.com — tons terrosos (areia, terracota, verde musgo), tipografia editorial Lora + DM Sans, ilustrações lineares, cards com cantos arredondados, layout orgânico mas estruturado.

STACK TÉCNICO

Frontend: Angular 17+ (standalone components, signals, lazy loading por módulo)
Backend: Node.js + Express REST API
Base de dados: MongoDB Atlas (Mongoose ODM)
Auth: JWT + Refresh Tokens + bcrypt
Real-time: Socket.io (notificações, fórum ao vivo)
Cache: Redis (taxas financeiras, preços de mercado)
Deploy: Frontend → Vercel, Backend → Render, DB → MongoDB Atlas
APIs externas:

Alpha Vantage / Finnhub → ações, ETFs
CoinGecko → cripto
ECB Data Portal → Euribor
Banco de Portugal API → TEAG
Steam Market API → preços CS2/skins
CSFloat API / Skinport API → skins CS2 detalhadas
