Ligar PayPal
Para obteres as credenciais, segue estes passos:
  1. Vai a developer.paypal.com e faz login com a tua conta PayPal.
  2. Clica em Apps & Credentials → Create App, dá um nome e clica Create.
  3. Copia o Client ID e o Client Secret (clica em "Show" para revelar).
  4. Cola os valores abaixo e clica Ligar PayPal.
⚠ O servidor usa o endpoint Sandbox por omissão. Para ligar a conta real, altera os URLs em server.js de api-m.sandbox.paypal.com para api-m.paypal.com.
